# SynthCore

**SynthCore** ist ein AI-gesteuerter Auto-Builder Agent, der Aufgaben automatisiert und Projekte selbstständig erstellt.  
Die Anwendung verwendet **OpenRouter API-Keys**, daher muss jeder Nutzer **einen eigenen Key** bereitstellen.

---

##  Funktionen

- Automatisches Erstellen von Projekten und Strukturen  
- KI-gesteuerte Entscheidungsfindung über OpenRouter  
- Unterstützt Anpassungen über Konfigurationsparameter  
- Ausgabe als ausführbare Datei (.exe) für Windows  

---

##  API-Setup

1. Registriere dich bei [OpenRouter](https://openrouter.ai)  
2. Erstelle einen persönlichen API-Key  
3. Trage den Key in der Konfigurationsdatei von SynthCore ein  
4. Starte die Anwendung

> Ohne gültigen API-Key funktioniert die KI-Funktionalität nicht.
